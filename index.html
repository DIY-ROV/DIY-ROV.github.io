DIY Underwater ROV

This project involves designing and building a functional underwater remotely operated vehicle (ROV) for exploration.

Challenges

Some challenges we faced included ensuring waterproofing, achieving precise control, and designing an efficient propulsion system.

Estimated Cost

The total cost of this project is estimated to be between $200 and $500, depending on the materials used.

Project Goal

The main goal of this project is to create a low-cost, functional underwater ROV that can explore aquatic environments.

Materials Needed

  • PVC pipes and connectors
  • Waterproof motors
  • Battery pack
  • Camera for underwater viewing
  • Control system (Arduino, Raspberry Pi, or similar)

Wiring & Electronics

The ROV's wiring system includes motor controllers, a power distribution board, and a waterproofed electronics enclosure.

Final Product

After assembling all components, we tested the ROV to assess its underwater movement, video clarity, and operational stability.
